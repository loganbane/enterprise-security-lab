Applied firewall rules, access control policies, and log analysis to protect services. Used basic penetration testing tools to identify vulnerabilities and strengthen system configurations.

# Enterprise Network Security Lab

## Overview
This project simulates an enterprise network environment to gain hands-on experience designing, deploying, and securing network services. Implemented DNS, LDAP, and Kerberos for centralized authentication and single sign-on (SSO).

## Core Services
- Primary and secondary DNS servers (ns1/ns2)
- LDAP directory services
- Kerberos authentication server
- Client systems integrated with SSO

## Skills Demonstrated
- Linux server administration
- Identity and access management
- Network security and troubleshooting
- Log analysis and service monitoring
